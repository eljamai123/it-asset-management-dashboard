# IT Asset Management Dashboard

## Overview

IT Asset Management Dashboard is a modern web application built with React, TypeScript, and Material UI for managing IT assets within an organization.

The application allows IT teams to track hardware inventory, manage support tickets, monitor users, and generate reports through an intuitive dashboard interface.

## Features

### Dashboard

* Overview of IT assets and statistics
* Asset status monitoring
* Ticket tracking
* Quick access to key metrics

### Inventory Management

* View all IT assets
* Add new assets
* Edit asset information
* Asset details panel
* Asset status tracking

### Ticket Management

* Create support tickets
* View ticket list
* Ticket details page
* Track ticket status
* Manage incidents and requests

### User Management

* Manage users
* View user information
* Assign assets to users

### Reports

* Asset reports
* Ticket reports
* Activity statistics

### Settings

* Application configuration
* Theme customization
* User preferences

## Technologies Used

* React.js
* TypeScript
* Vite
* Material UI (MUI)
* React Context API

## Installation

Clone the repository:

```bash
git clone https://github.com/eljamai123/it-asset-management-dashboard.git
```

Navigate to the project directory:

```bash
cd it-asset-management-dashboard
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
